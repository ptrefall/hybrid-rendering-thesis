
\begin{abstract}
This work investigates a GPU only approach for rendering images, where both raster and ray-tracing strategies are used in hybrid, in order to improve visual quality while maintaining an interactive or near real-time performance.

\textbf{Keywords:} Hybrid rendering, Ray tracing, Deferred shading, Real-time, GPU, Raster, OpenGL, GLSL, OptiX, CUDA
\end{abstract}
